PHOENIX NEST
Phoenix Nest is an playable "metagame" of my work (a showcase within a showcase), made with the Unity game engine. I thought this would be a good opportunity to work on this over the 2021/22 holiday period as a way to learn the software while staying busy. I've made basic games with the engine before, but I want to see how far I can push it.
